# SPA RAILS-ANGULAR

### Descripción 
- Ejemplo de SPA (Single Page Application) en Angular utilizando Rails en el backend(API).

### Instalación
```bash
bundle install
rails db:create
rails db:migrate
rails db:seed
rails server
```
